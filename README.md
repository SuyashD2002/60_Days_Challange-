## 🎯 The 60-Day Vacation Sprint

Instead of taking time off during the 2-month university break, I committed to a structured daily regime to master key technical competencies expected in the robotics and automation industries.

### 🚀 Daily Focus & Core Stack
- **C++:** Modern syntax, OOP, STL, memory management, and data structures.
- **ROS 2:** Robot architecture, publisher/subscriber nodes, services, TF2, URDF, and navigation.
- **MATLAB & Simulink:** Control systems design, Stateflow, PID tuning, and physical modeling.
- **German Language:** Daily vocabulary, grammar, and speaking practice (aiming for conversational fluency).

---

## 📊 Dashboard Capabilities

I built this dashboard to keep myself accountable every single day of the challenge:

- 📅 **Interactive 60-Day Matrix:** Daily checklists, task tracking, and engineering logs.
- ⏱️ **Study Analytics:** Real-time tracking of focused study hours and learned German vocabulary.
- ⚡ **Dynamic Skill Tree:** Visual proficiency tracking across my core technical stack using Chart.js.
- 📝 **LinkedIn Progress Updates:** Automatic status summary generator to share daily and weekly milestones.
- 💾 **100% Offline Persistence:** Uses browser LocalStorage to save progress without needing external servers.
